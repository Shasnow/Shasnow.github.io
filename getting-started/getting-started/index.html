开始使用

我们将从下载开始，一步步引导您学会SRA的基本使用操作。

如果对使用过程/教程遇到任何问题，请首先怀疑开发者。提交issue 或前往官方群聊中反馈。

下载

Section titled “下载”

前往下载页面选择渠道进行下载

添加信任

Section titled “添加信任”

SRA拥有多种敏感动作（包括但不限于获取管理员权限、读写注册表、操作屏幕）而导致容易被安全软件误报隔离。 在运行软件前，可以将SRA的安装目录添加到安全软件的信任列表中以避免被误杀。 此处展示如何在Windows Defender中添加信任：

  1. 打开Windows Defender

    如果链接无效，请手动打开Windows Defender

    如果安装了其他的防病毒软件，请先打开定期扫描。

  2. 打开病毒和威胁防护设置 - 管理设置

  3. 点击排除项 - 添加或删除排除项

  4. 点击排除项 - 选择对应目录

  5. 若已经安装其他软件，再次关闭定期扫描。

如果您拥有其他安全软件，您可能需要额外配置。

即使已经安装其它杀毒软件（如：火绒、360极速版），Windows Defender 防病毒功能仍可能会不定时开启， 这可能导致您的SRA.exe突然消失。因此，您必须确保以上目录被Windows Defender排除。

安装&启动

Section titled “安装&启动”

作为一个绿色程序，SRA不需要安装，直接解压缩后运行SRA.exe即可。

自 SRA 2.14 开始，运行 SRA 需要 .NET 10.0 运行时支持。 如果您的系统中没有安装该运行时，您会在首次运行时收到提示。请 点击该链接 下载并安装 .NET 10.0。

推荐为SRA创建快捷方式，但不要修改源程序的位置。

运行时需要以管理员身份运行。

启动成功后，将同时弹出公告及主界面。公告将在下次更新前不再弹出。

运行前配置

Section titled “运行前配置”
  • 📺 分辨率要求：
    • 如果你的屏幕分辨率大于1920x1080(例如2560x1440)，请将游戏调整至1920x1080窗口模式下运行，否则可能出现识别错误！
    • 如果你的屏幕分辨率等于1920x1080，请将游戏调整至1920x1080全屏模式下运行，否则可能出现识别错误！
    • 如果你的屏幕分辨率小于1920x1080，请等待后续版本更新，目前可能无法正常使用！
  • 🖱️ 操作提示：执行任务时不要进行其他键鼠操作！

任务配置

Section titled “任务配置”

在侧边栏选中任务选项卡，可以看到任务配置界面。勾选任务名称左侧的复选框以启用对应任务。

启动游戏

Section titled “启动游戏”

在这个任务中，可以设置与启动游戏相关的任务。

  1. 渠道：选择登录渠道，目前已经支持官服、b服以及国际服的启动。

  2. 游戏路径： 如果启用了“使用全局设置”，此处的设置将被全局设置覆盖，SRA会使用全局设置中的游戏路径来启动游戏。 否则，SRA会使用此处设置的游戏路径来启动游戏。选择路径时，应该选择“游戏本体”，而不是“启动器”

    提示

    一个有效的游戏路径示例：C:\Program Files\miHoYo Launcher\games\Star Rail Game\StarRail.exe

  3. 自动登录：决定是否在启动后进行登录。 如果启用，则必须填入您的账号密码以便SRA进行登录；反之，则必须保证您处于登录状态。 勾选自动登录表明您已阅读并同意《用户协议》和《隐私政策》

  4. 总是重新登录：决定是否每次都进行重新登录。如果启用，则每次启动游戏后都会登出当前账号并重新登录。

清体力

Section titled “清体力”

在这个任务中，可以设置是否补充体力、通过何种方式补充、挑战的关卡等。

  • 任务清单：点击右侧配置按钮，可以添加、删除、修改任务列表中的项。

    如果要删除任务列表中的某一项，可以选中该项然后右键点击列表，选择删除。

    当SRA执行到清开拓力这一任务时，会按照任务列表从上往下依次执行。

    任务列表的内容是可以重复的，他们的顺序取决于您添加的顺序。

    每次执行均从头开始执行，且执行完成后不会删除已经执行的任务。也就是下一次依然复用同样的清单。

    点击添加按钮，可以添加新的任务项。 对于每类副本，在左侧选择关卡、次数等信息，然后点击 确认 将它添加进任务列表中。

    可以添加多种副本。当体力不足而未补充体力时，此后的战斗均不会执行。

    • 自动检测：启用后，将不允许手动修改次数，SRA会根据当前体力和关卡消耗自动计算可挑战次数。 可以添加多个自动检测项，它们会平分当前体力。 可以将自动检测与手动添加的关卡混合使用。手动添加的关卡总会会优先执行。

    提示

    示例：当前体力300，关卡A消耗60体力，关卡B消耗30体力。

    • 添加关卡A，次数设置为自动检测
    • 添加关卡B，次数设置为自动检测 计算过程：
    • 总体力消耗：60+30=90体力
    • 每个关卡可挑战次数：当前体力/关卡消耗=300/90=3次
    • 剩余体力：300-3*90=30体力
    • 则SRA会为关卡A、B各分配3次挑战次数，同时，为关卡B额外分配1次挑战次数以利用剩余体力。
    • 结果：关卡A可挑战3次，关卡B可挑战4次。

  • 补充体力：每次补充60点，即取出60后备开拓力或使用一罐燃料或消耗星琼。

  • 使用支援角色：决定是否使用助战角色进行挑战。SRA只会使用支援列表中第一个角色。

  • 启用培养目标：如果启用，SRA会忽略任务清单中的关卡设置，并在游戏内根据培养目标界面中设置的目标自动分配体力。

  • 启用多倍活动检测： 该功能将检测当前是否有花藏繁生、异器盈界等活动，并优先刷对应的副本。 开启后，在清体力时将跳转到活动页面检测当前是否有多倍活动，如果有则优先刷对应的副本。 实现方式是将对应的副本添加到自动检测的副本列表头部，因此培养目标或手动添加的自动检测任务仍然有效。

领取奖励

Section titled “领取奖励”

在这个任务中包含了所有的每日奖励，如每日实训，无名勋礼。部分是默认启用的。

还可以领取兑换码奖励，勾选此项，并在兑换码输入框内输入兑换码，可以输入多个兑换码，兑换码之间用空格或换行分隔。

旷宇纷争-差分宇宙

Section titled “旷宇纷争-差分宇宙”

目前只支持差分宇宙-周期演算 刷第一关，即通过反复刷取第一关来提升等级。

每周执行20次即可完成每周积分奖励。

使用执行前，你需要确保已经完成至少一次差分宇宙以清除所有新手提示。

旷宇纷争-货币战争

Section titled “旷宇纷争-货币战争”

支持标准博弈和超频博弈或刷开局，可选择最高难度、最低难度或当前难度。分为全程模式和接管模式。

  1. 开拓者名称：无论哪种模式，都需要填写 ‘开拓者名称’ 。如果您的开拓者名称中包含特殊字符或生僻字，可能会导致识别失败，请尽量使用常见字符。
  2. 类型：选择执行的货币战争类型，标准博弈、超频博弈或刷开局。
  3. 难度：选择执行的货币战争难度，最高难度、最低难度或当前难度。
  4. 攻略：选择执行的货币战争攻略，关于如何创建或导入攻略请参阅货币战争自定义攻略。
  5. 运行次数：设置执行的轮数。如果为刷开局，则表示尝试次数；如果为标准博弈或超频博弈，则表示重复执行的轮数。

准备好之后，记得勾选货币战争复选框和旷宇纷争复选框以启用该任务。

全程模式

Section titled “全程模式”

提示

这不是一个可供选择的任务模式，仅为SRA在指定条件下的运行逻辑。

在这个模式下，SRA会从头到尾完整地执行货币战争的所有步骤，包括选择难度、选择阵容、战斗等。

要以全程模式执行货币战争任务，您必须处于货币战争的主界面或大世界界面。

接管模式

Section titled “接管模式”

提示

这不是一个可供选择的任务模式，仅为SRA在指定条件下的运行逻辑。

在这个模式下，SRA会接管已经开始的货币战争任务，从当前进度继续执行。

要以接管模式执行货币战争任务，您必须处于货币战争的准备阶段界面。

刷开局

Section titled “刷开局”

在货币战争任务中，“类型”下拉菜单中选择“刷开局”

  • Boss名称: 在此处填写你想要刷出的Boss名称，例如 巨鹿生物制药;增熵能源集团;金血记忆体联盟，Boss名称以货币战争中显示的Boss名称为准。 使用 ;（中英文均可）分隔每个Boss，分别代表期望在第一位面刷出“巨鹿生物制药”，第二位面刷出“增熵能源集团”，第三位面刷出“金血记忆体联盟”。 如果对应位面没有期望的Boss，将其留空，例如 ;;金血记忆体联盟表示期望在第三位面刷出“金血记忆体联盟”，而第一、二位面则没有特定要求。
  • Boss词条: 在此处填写你想要刷出的Boss词缀名称，例如 变宝为废，使用空格分隔多个词缀。 也可以在词缀名称前添加 !（中英文均可） 来表示不想要的词缀，例如 !变宝为废 表示不想要“变宝为废”这个词缀
  • 投资环境: 在此处填写你想要刷出的投资环境名称，例如 长线利好 轮岗，使用空格分隔多个环境。 也可以在环境名称前添加 ?（中英文均可） 来表示可选环境，例如 ?长线利好 表示如果有“长线利好”这个环境就更好，但没有也无所谓
  • 投资策略: 在此处填写你想要刷出的投资策略名称，例如 砂里淘金，使用 ;（中英文均可）分隔不同位面的策略， 例如 轮回不止；环保大使叽米表示第一位面期望刷出“轮回不止”，第二位面期望刷出“环保大使叽米”。 轮回不止；或轮回不止表示第一位面期望刷出“轮回不止”，第二位面没有特定要求。 ;环保大使叽米表示第一位面没有特定要求，第二位面期望刷出“环保大使叽米”。
  • 特别地，所有的匹配都是双向子字符串匹配，你可以通过拼接不同的内容来实现更复杂的匹配。 例如 白银投资 溜佩佩 招财狗； 可以表示第一位面期望刷出“白银投资”或“溜佩佩“或“招财狗”，第二位面没有特定要求。 轮回;环保大使 等同于 轮回不止；环保大使叽米。

结束后

Section titled “结束后”

用于控制结束后的操作，包含以下任务：

  1. 登出账号: 将会在游戏菜单右下角点击登出账号按钮，登出当前账号。
  2. 退出SRA: 将会结束SRA程序。
  3. 退出游戏: 将会关闭游戏窗口。
  4. 关机或休眠: 将会在60秒后关机或休眠计算机。

开始、停止任务

Section titled “开始、停止任务”

在主界面的底部或任务界面的顶部，可以看到操作面板。点击面板最右侧的按钮即可开始执行任务。

面板最右侧的按钮有3种模式，通过展开右侧箭头来切换：

  • 启动当前配置：执行当前任务配置
  • 启动全部配置：从列表的第一个开始，执行所有已保存的任务配置，适用于多账号托管
  • 仅保存数据：仅保存当前配置但不执行任务

我们会在进阶操作中介绍如何创建更多配置以便为不同的账号运行不同的任务。

一但点击任务正确运行，按钮会变为停止，表示任务正在执行中。

您可以转到控制台选项卡查看任务的执行日志。

我们会在进阶操作中介绍日志过滤器。

如果需要中断任务，可以点击停止，或按下快捷键 F9 ，任务会立即停止执行。

开始执行任务时，SRA会尝试让游戏窗口处于焦点状态，此时不建议人工干预，除非需要中断任务。
