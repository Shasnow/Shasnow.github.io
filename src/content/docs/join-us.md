## 参与贡献
如果你想参与 StarRailAssistant 的维护，欢迎通过下面的方式加入我们。

### 你可以做什么

- 提交 Bug 报告，帮助我们更快定位问题。
- 提交 Pull Request，修复错误或补充功能。
- 改进文档，让新用户更容易上手。
- 参与测试，反馈不同设备和环境下的表现。

### 参与方式

1. 在 GitHub 上打开仓库并查看现有 Issue。
2. 如果你发现问题，优先提交一个清晰的 Issue，说明复现步骤和截图。
3. 如果你愿意直接修复，可以 Fork 仓库后提交 PR。
4. 如果你只是想帮忙，也欢迎补充使用反馈和建议。

### 我们更欢迎这些贡献

- 文档补充与纠错
- 新功能提案
- 图片识别相关优化
- 不同分辨率、不同系统环境的测试反馈

### 联系与协作

如果你已经在参与开发，建议保持提交内容尽量聚焦，避免一个 PR 同时改太多无关内容。
这样更方便我们快速 review 和合并。
