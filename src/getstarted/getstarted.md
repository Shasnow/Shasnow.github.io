# 开始使用 StarRailAssistant

我们将从下载开始，一步步引导您学会SRA的基本使用操作。

> 如果对使用过程/教程遇到任何问题，请首先怀疑开发者。提交[issue](https://github.com/Shasnow/StarRailAssistant/issues)
> 或前往官方群聊中反馈。

## 下载

前往[下载页面](https://starrailassistant.top/SRA/download.html)选择渠道进行下载

## 添加信任

SRA拥有多种敏感动作（包括但不限于**获取管理员权限**、**读写注册表**、**操作屏幕**）而导致容易被安全软件误报隔离。
在运行软件前，可以将SRA的安装目录添加到安全软件的信任列表中以避免被误杀。
此处展示如何在**Windows Defender**中添加信任：

1. 打开[Windows Defender](ms-settings:windowsdefender)

   如果链接无效，请手动打开**Windows Defender**

   如果安装了其他的防病毒软件，请先打开*定期扫描*。

   ![添加信任步骤1](/img/trust/truststep1.png)

2. 打开**病毒和威胁防护**设置 - **管理设置**

   ![添加信任步骤2](/img/trust/truststep2.png)

3. 点击**排除项** - **添加或删除排除项**

   ![添加信任步骤3](/img/trust/truststep3.png)

4. 点击**排除项** - 选择对应目录

   ![添加信任步骤4](/img/trust/truststep4.png)

5. 若已经安装其他软件，再次关闭*定期扫描*。

:::warning
如果您拥有其他安全软件，您可能需要额外配置。

即使已经安装其它杀毒软件（如：火绒、360极速版），Windows Defender 防病毒功能仍可能会不定时开启，
这可能导致您的SRA.exe突然消失。因此，您必须确保以上目录被Windows Defender排除。
:::

## 安装&启动

作为一个绿色程序，SRA不需要安装，直接解压缩后运行**SRA.exe**即可。

:::warning
自 SRA 2.0 开始，运行 SRA 需要 `.NET 8.0` 运行时支持。
如果您的系统中没有安装该运行时，您会在首次运行时收到提示。请前往 [.NET 官网](https://dotnet.microsoft.com/en-us/download/dotnet)
下载并安装 `.NET 8.0`。

:::

推荐为SRA创建快捷方式，但不要修改源程序的位置。

运行时需要以管理员身份运行。

![SRA.exe](/img/use/software.png)

启动成功后，将同时弹出公告及主界面。点击确认可关闭公告，点击不再提醒可关闭公告，直到下次更新前将不再弹出。
![SRA2.0主界面](/img/use/softwaremain2.png)

## 任务配置

![任务配置](/img/use/mission2.png)

在侧边栏选中**任务**选项卡，可以看到任务配置界面。勾选任务名称左侧的复选框以启用对应任务。

### 启动游戏

在这个任务中，可以设置与启动游戏相关的任务。

1. 渠道：选择登录渠道，目前已经支持官服和b服的启动。
2. 请在选择文件路径时选择**游戏本体**的路径

:::tip
一个有效的游戏路径示例：`C:\Program Files\miHoYo Launcher\games\Star Rail Game\StarRail.exe`
:::

3. 自动登录：决定是否在启动后进行登录。如果启用，则必须填入您的账号密码以便SRA进行登录；反之，则必须保证您处于登录状态。勾选自动登录表明您已阅读并同意《用户协议》和《隐私政策》
4. 总是重新登录：决定是否每次都进行重新登录。如果启用，则每次启动游戏后都会登出当前账号并重新登录。

### 清体力

在这个任务中，可以设置是否补充体力、通过何种方式补充、挑战的关卡等。

补充体力：每次补充60点，即取出60后备开拓力或使用一罐燃料或消耗星琼。

使用支援：决定是否使用助战角色进行挑战。SRA只会使用支援列表中第一个角色。

重新编队：如果启用，SRA会在同一关卡连续挑战时重新选择编队，通常和使用支援一起开启。

编队检查：仅影响 饰品提取 ，当启用时，SRA会检查当前编队是否满员，若未满员则重新使用1号编队。

战技入场：仅影响 饰品提取 ，当启用时，SRA会在战斗开始前依次使用1~4号角色的战技，然后使用最后一位角色进入战斗。

对于每类副本，勾选则代表启用，可以设置指定关卡，连战次数、挑战次数等。

可以勾选多类副本。当体力不足而未补充体力时，此后的战斗均不会执行。

![清开拓力配置](/img/use/msc2.png)

在左侧选择关卡、次数等信息，然后点击 `+` 将它添加进任务列表中。

自动检测：启用后，将不允许手动修改次数，SRA会根据当前体力和关卡消耗自动计算可挑战次数。
可以添加多个自动检测项，它们会平分当前体力。
可以将自动检测与手动添加的关卡混合使用。手动添加的关卡总会会优先执行。

:::tip
示例：当前体力300，关卡A消耗60体力，关卡B消耗30体力。

- 添加关卡A，次数设置为自动检测
- 添加关卡B，次数设置为自动检测
- 则SRA会为关卡A、B各分配150体力，分别可挑战2次和5次。

:::

如果要删除任务列表中的某一项，可以选中该项然后右键点击列表，选择删除。

当SRA执行到清开拓力这一任务时，会按照任务列表从上往下依次执行。

任务列表的内容是可以重复的，他们的顺序取决于您添加的顺序。

每次执行均从头开始执行，且执行完成后不会删除已经执行的任务。也就是下一次依然复用同样的清单。

### 领取奖励

在这个任务中包含了所有的每日奖励，如每日实训，无名勋礼。部分是默认启用的。

还可以领取兑换码奖励，勾选此项，并在兑换码输入框内输入兑换码，可以输入多个兑换码，兑换码之间用空格或换行分隔。

### 模拟宇宙

目前只支持差分宇宙-周期演算 刷单层，即通过反复刷取第一层来提升等级。

要执行差分宇宙任务，首先需要手动进入差分宇宙的主界面。

![差分宇宙主界面](/img/use/universehome.png)

务必在此处开始执行，不在此界面将导致无法正常执行。

:::warning
使用执行前，你需要确保已经完成至少一次差分宇宙以清除所有新手提示。
:::

### 货币战争

支持标准博弈和超频博弈或刷开局，可选择最高难度或最低难度。分为全程模式和接管模式。

无论哪种模式，都需要填写 '开拓者名称' 。如果您的开拓者名称中包含特殊字符或生僻字，可能会导致识别失败，请尽量使用常见字符。

准备好之后，记得勾选货币战争复选框和旷宇纷争复选框以启用该任务。

#### 全程模式

在这个模式下，SRA会从头到尾完整地执行货币战争的所有步骤，包括选择难度、选择阵容、战斗等。

要以全程模式执行货币战争任务，您必须处于货币战争的主界面或大世界界面。

![货币战争主界面](/img/use/currency_wars_main.png)

#### 接管模式

在这个模式下，SRA会接管已经开始的货币战争任务，从当前进度继续执行。

要以接管模式执行货币战争任务，您必须处于货币战争的准备阶段界面。

![货币战争准备界面](/img/use/currency_wars_ready.png)

#### 刷开局

在货币战争任务中，"类型"下拉菜单中选择"刷开局"，默认使用最高难度进行刷开局
投资环境: 在此处填写你想要刷出的投资环境名称，例如“长线利好”，“轮岗”，使用空格分隔多个环境
投资策略: 在此处填写你想要刷出的投资策略名称，例如“砂里淘金”，使用空格分隔多个策略
当同时填写了投资环境和投资策略时，当且仅当两者均符合时才视为成功刷出
策略阶段: 所选的投资策略会在哪一阶段出现
最大尝试轮数: 设置最大尝试刷出轮数，超过该轮数后任务将停止并失败，每一次开局记为一轮，最高可设置为99999轮

### 结束后

用于控制结束后的操作，包含以下任务：

1. 登出账号
2. 退出SRA
3. 退出游戏
4. 关机或休眠

## 开始、停止任务

在主界面的底部或任务界面的顶部，可以看到操作面板。点击面板最右侧的按钮即可开始执行任务。

面板最右侧的按钮有3种模式，通过展开右侧箭头来切换：

- 启动当前配置：执行当前任务配置
- 启动全部配置：从列表的第一个开始，执行所有已保存的任务配置，适用于多账号托管
- 仅保存数据：仅保存当前配置但不执行任务

我们会在[进阶操作](/getstarted/advance)中介绍如何创建更多配置以便为不同的账号运行不同的任务。

一但点击任务正确运行，按钮会变为**停止**，表示任务正在执行中。

您可以转到控制台选项卡查看任务的执行日志。

我们会在[进阶操作](/getstarted/advance#日志过滤器)中介绍日志过滤器。

如果需要中断任务，可以点击**停止**，或按下快捷键 `F9` ，任务会立即停止执行。

![操作按钮](/img/use/operation2.png)

开始执行任务时，SRA会尝试让游戏窗口处于焦点状态，此时不建议人工干预，除非需要中断任务。

:::tip
现已支持按下 `F9` 快捷键停止任务。
:::
